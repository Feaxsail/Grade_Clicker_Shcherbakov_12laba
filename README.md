# Grade Clicker — Лабораторная работа №12

## 📱 Описание проекта

**Grade Clicker** — интерактивное Android-приложение, демонстрирующее работу жизненного цикла Activity (Activity Lifecycle) и сохранение состояния при изменении конфигурации устройства.

### Механика приложения:
- Пользователь кликает по изображению, набирая баллы
- За каждый клик начисляется **+5 баллов**
- При достижении определённого порога баллов изменяется отображаемая оценка и изображение
- Приложение сохраняет состояние при повороте экрана

### Система оценок:
| Баллы | Оценка | Сообщение |
|-------|--------|-----------|
| 0-49 | Неудовлетворительно | "Продолжайте работать!" |
| 50-69 | Удовлетворительно | "Вы получили зачёт!" |
| 70-89 | Хорошо | "Отличная работа!" |
| 90-100 | Отлично | "Вы успешно сдали курс!" |

## 🎯 Цель работы

Изучить жизненный цикл Activity в Android и научиться правильно обрабатывать переходы между состояниями:
- Освоить callback-методы жизненного цикла (onCreate, onStart, onResume, onPause, onStop, onDestroy, onRestart)
- Научиться использовать логирование (Logcat) для отслеживания состояния приложения
- Понять проблему потери данных при изменении конфигурации (поворот экрана)
- Изучить разницу между `remember` и `rememberSaveable` для сохранения состояния

## ❓ Контрольные вопросы

### 1. Какие три метода вызываются при запуске приложения?

При запуске приложения вызываются три метода в следующем порядке:
1. **`onCreate()`** — Activity создаётся, выполняется инициализация
2. **`onStart()`** — Activity становится видимой для пользователя
3. **`onResume()`** — Activity получает фокус и становится активной
